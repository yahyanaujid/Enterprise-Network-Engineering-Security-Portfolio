# Enterprise Network Engineering & Security Portfolio

A technical portfolio demonstrating the design and implementation of secure, multi-branch network infrastructures for corporate and e-commerce environments.

## 🛠️ Technical Stack
* Simulation Tool: Cisco Packet Tracer.
* Routing & Switching: Inter-VLAN Routing (Router-on-a-Stick), VTP (VLAN Trunking Protocol), and Dynamic Routing.
* Security & Hardening: Port Security, Standard/Extended ACLs (Access Control Lists), and DHCP Snooping.
* Network Logic: VLSM (Variable Length Subnet Masking) for optimized IP allocation.
* Services: Centralized DHCP, Wireless Security (WPA2), and DNS configuration.

## 🚀 Key Projects

### 1. Corporate Network Segmentation 
Designed a high-performance departmentalized network focused on traffic isolation and logical management.
* VLAN Architecture: Implemented VTP to maintain consistent departmental segmentation (IT, HR, Finance) across a multi-switch environment.
* Inter-VLAN Routing: Configured Router-on-a-Stick to allow controlled communication between isolated subnets while maintaining security boundaries.
* Access Layer Security: Deployed Port Security to mitigate unauthorized physical device connections and MAC address spoofing.
* WAN Connectivity: Established secure links between regional LANs (Penang/Thailand) and a centralized DMZ for shared resources.

### 2. NeuroByte Multi-Branch Infrastructure 
Developed a scalable network for an expanding e-commerce startup (NeuroByte) connecting four major branches (Ipoh, Melaka, Kedah, Perlis).
* Branch Logic (Ipoh): Led the design of the Ipoh Branch infrastructure, implementing localized DHCP and static IP addressing for critical servers.
* Dynamic Routing: Configured a  Full Mesh Topology  using dynamic routing protocols to ensure high availability and automatic failover between all four branch offices.
* Wireless Security: Secured internal branch Wi-Fi using  WPA2 encryption , ensuring mobile warehouse devices could connect safely to the internal network.
* Infrastructure Optimization: Utilized  VLSM to create a hierarchical IP scheme that supports rapid branch expansion without address exhaustion.
